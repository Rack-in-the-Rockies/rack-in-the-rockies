# Copy style

Never use em dashes (—) or en dashes (–) in any user-facing copy. Use a comma,
a colon, a period, or a plain hyphen instead. This applies to page text, button
labels, metadata descriptions, and data files. Time ranges use a hyphen:
"4:45 - 8:00 PM".
